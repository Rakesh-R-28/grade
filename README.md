📘 Student Grade Analyzer

This is a simple Python project that calculates the average marks of a student and assigns a grade based on the result.

🚀 Features

Takes marks for 3 subjects

Calculates the average

Displays the grade (A, B, C, or Fail)

🧠 Concepts Used

Functions

Classes (OOP basics)

Conditional statements


📊 Grade Criteria

90 and above → A

75 to 89 → B

50 to 74 → C

Below 50 → Fail

💡 Purpose

This project is created to practice Python functions and basic programming logic.
